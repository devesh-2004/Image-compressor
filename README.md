# Image Compressor

A free, fast, **100% client-side** image compressor. Drag in a JPG, PNG, or
WebP and shrink it right in your browser — nothing is ever uploaded to a server.

Built with **Next.js (App Router) + React + TypeScript + Tailwind CSS**, using
[`browser-image-compression`](https://www.npmjs.com/package/browser-image-compression)
for the actual work. Deploys to Vercel with zero config.

## Features

- Drag-and-drop **or** file-picker upload (JPG, PNG, WebP)
- Compression runs entirely in the browser — no backend, no API, no cost
- Live **original vs. compressed** preview with sizes and % reduction
- Quality slider to control the compression level (re-compresses live)
- One-click download with the correct file extension
- Friendly error handling for unsupported types and oversized files (20 MB cap)
- Clean, responsive UI with a subtle loading state during compression

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```
